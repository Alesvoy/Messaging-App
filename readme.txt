Paso 1:
Para poder visualizar la aplicacion necesitamos descargar Node.js conjunto npm en la maquina y posteriormente crear una carpeta que se llame messagingapp. Descargamos los archios de github y los me temos a la carpeta. Abrimos una terminal como administrador y nos posicionamos en la carpeta recientemente creada, despues corremos el comando npm init (Dejamos la configuracion predeterminada por npm).

Paso 2: Nos volvemos a posicionar en la terminal y todavia dentro de la carpteta corremos los siguietes comandos: 

npm install socket.io --save
npm install nodemon --save

Paso 3: Inicializamos el servidor corriendo el comando:

npm run devStart

Paso 4: Abrimos cualquier navegador, recomiendo utlizar google chrome. Abrimos 2 ventanas y en ambas abrimos (Ctrl + o) el archivo html dentro de nuestra carpeta y probamos la aplicacion.
